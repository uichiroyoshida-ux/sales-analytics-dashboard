# 📊 Sales Analytics Dashboard

An interactive business intelligence dashboard built with Python and Streamlit.

## Features
- Real-time filtering by region, category, and time period
- Key performance indicators (Revenue, Profit, Orders, Margin)
- Sales trend analysis
- Regional and customer segment breakdown
- Profitability analysis by sub-category
- Top products and shipping insights

## Tech Stack
- Python
- Pandas
- Streamlit
- Plotly
